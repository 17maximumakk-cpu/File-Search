File Search Suite  мощный поиск файлов
Коллекция из семи полнофункциональных реализаций поисковика файлов на разных языках программирования. Каждая версия включает:

📁 Рекурсивный поиск по указанной директории (и подпапкам)

🔤 Поиск по имени файла (маска / регулярное выражение)

📄 Поиск по содержимому (текст в файлах, поддерживаются .txt, .md, .py, .html и др.)

🧹 Фильтры по расширению, размеру (мин/макс), дате изменения

⚡ Многопоточность / асинхронность для ускорения (в GUI/CLI)

📊 Прогресс-бар и отмена поиска

💾 Сохранение результатов в CSV / JSON

🎨 Цветной вывод в консоли (для CLI-версий)

📌 Исключение папок (например, .git, node_modules, __pycache__)

Языки и технологии
Язык	Интерфейс	Библиотеки / фреймворки	Особенности
Python	CLI + Tkinter GUI	os, fnmatch, threading	Графический интерфейс с прогресс-баром, многопоточность, экспорт CSV
JavaScript	Веб (браузерная)	File System Access API, idb (опционально)	Поиск в виртуальной файловой системе (песочница) / Node.js вариант
Go	CLI	filepath.Walk, sync, regexp	Высокая производительность, конкурентный обход
Rust	CLI	walkdir, ignore, rayon (опционально)	Безопасность, скорость, цветной вывод
Java	Swing GUI	java.nio.file, ExecutorService	Десктопное приложение с JTable, фильтры
C#	WinForms / WPF	System.IO, Task	Асинхронный поиск, DataGridView, экспорт в Excel
PHP	CLI / Веб	RecursiveDirectoryIterator	Простой запуск из браузера или командной строки
